CleanrApp — نظام إدارة خدمات التنظيف
هل سبق أن فكّرت كيف تُدار مئات طلبات التنظيف في الفنادق والمجمعات السكنية يومياً؟

بنيت CleanrApp، منصة ويب متكاملة تحوّل فوضى تنسيق خدمات التنظيف إلى سير عمل سلس ومتتبَّع لحظة بلحظة.

ما يميّز هذا المشروع
🏗️ معمارية متعددة الأدوار — أربع واجهات مستقلة (مدير، مشرف، عامل، عميل) كل منها مصمم لاحتياجات صاحبه بالضبط.

⚡ تتبع فوري — إشعارات real-time بـ SignalR تُعلم كل طرف فور تغيير حالة الطلب، من لحظة الإنشاء حتى الاعتماد النهائي.

📸 توثيق مرئي — العمال يرفعون صور التنظيف مباشرة من الموقع، والمشرف يراجعها قبل الاعتماد.

🔒 صلاحيات دقيقة — Role-Based Access Control يضمن أن كل مستخدم لا يرى إلا ما يخصّه.

📊 تقارير ذكية — لوحة تحكم تعرض أداء كل عامل، رصيد زيارات العملاء، والإحصائيات اليومية والشهرية.

التقنيات المستخدمة
.NET 9 · ASP.NET Core MVC · Entity Framework Core · SQLite · SignalR · Bootstrap 5 RTL · Docker · Render.com

من فكرة إلى إنتاج
المشروع مبني بالكامل بـ Clean Architecture ومنشور على السحابة بـ Docker، مع قاعدة بيانات مستمرة وواجهة عربية RTL احترافية.

فكرة بسيطة، تنفيذ محكم — هكذا تُبنى الأنظمة.
